# WeGrow - 성장을 위한 목표 관리 앱

## 자기소개

안녕하세요! 저는 Flutter를 활용한 모바일 앱 개발에 관심이 있는 개발자입니다. 사용자의 성장과 목표 달성을 돕는 의미 있는 앱을 만들고자 WeGrow를 개발하게 되었습니다.

## 아이디어를 선정한 이유

### 🎯 문제 인식
- 많은 사람들이 목표를 세우지만 체계적으로 관리하지 못함
- 목표 달성 과정에서의 성찰과 회고가 부족함
- 혼자서만 목표를 추진하다 보니 지속성이 떨어짐
- 단순한 할 일 관리가 아닌, 진정한 성장을 위한 도구가 필요함

### 💡 해결 방안
- **계층적 목표 관리**: 월간 → 주간 → 일간 목표로 체계적 관리
- **회고 시스템**: 목표 완료 시 다양한 방식의 회고를 통한 성찰
- **소셜 기능**: 친구들과 목표를 공유하며 동기부여
- **시간 추적**: 실제 투입 시간을 기록하여 현실적인 목표 설정

## 아이디어 기능 목록

### 📋 핵심 기능
1. **계층적 목표 관리**
   - 월간 목표 → 주간 목표 → 일간 목표 구조
   - 상위 목표 완료를 위한 하위 목표 자동 체크

2. **회고 시스템**
   - 한 줄 회고: 간단한 성찰
   - KPT 회고: Keep/Problem/Try 구조
   - 이모지 회고: 감정 기반 평가

3. **시간 추적**
   - 일간 목표별 시간 기록
   - 실시간 초 단위 업데이트
   - 목표 달성률 계산

4. **소셜 기능**
   - 친구 추가/관리
   - 목표 공개 설정 (비공개/친구공개/전체공개)
   - 피드에서 친구들의 목표 확인
   - 좋아요, 댓글, 공유 기능

5. **프로필 및 통계**
   - 목표 완료율 통계
   - 연속 달성 일수 (스트릭)
   - 회고 히스토리

### 🎨 UI/UX 특징
- **다크 테마**: 집중력 향상을 위한 검정색 기반 디자인
- **직관적 네비게이션**: 하단 탭으로 쉬운 접근
- **애니메이션**: 목표 완료 시 폭죽 애니메이션
- **반응형 디자인**: 다양한 화면 크기 지원

## 구현한 내용 설명

### 🏗️ 아키텍처
```
lib/
├── main.dart                 # 앱 진입점
├── services/                 # 비즈니스 로직
│   ├── goal_service.dart     # 목표 관리 서비스
│   └── local_storage_service.dart # 로컬 저장소
└── presentation/             # UI 레이어
    ├── pages/               # 페이지들
    │   ├── home/           # 홈 탭
    │   ├── goal/           # 목표 관리
    │   ├── feed/           # 소셜 피드
    │   ├── profile/       # 프로필
    │   └── reflection/    # 회고
    └── widgets/            # 재사용 위젯
```

### 🔧 주요 구현 사항

#### 1. 목표 관리 시스템
- **계층적 구조**: 월간 → 주간 → 일간 목표의 자동 연동
- **진행률 계산**: 하위 목표 완료에 따른 상위 목표 진행률 자동 업데이트
- **완료 조건**: 모든 하위 목표 완료 시에만 상위 목표 완료 가능

#### 2. 회고 시스템
- **3가지 회고 타입**: 사용자 선택에 따른 다양한 회고 방식
- **태그 시스템**: 회고에 태그를 추가하여 카테고리화
- **히스토리 관리**: 날짜별 회고 목록 및 검색 기능

#### 3. 소셜 기능
- **친구 시스템**: 사용자 간 친구 추가/삭제
- **공개 설정**: 목표별 공개 범위 설정
- **피드 시스템**: 친구들의 목표와 회고 확인
- **상호작용**: 좋아요, 댓글, 공유 기능

#### 4. 데이터 관리
- **로컬 저장소**: SharedPreferences를 활용한 오프라인 데이터 저장
- **테스트 데이터**: 다양한 사용자와 목표로 구성된 샘플 데이터
- **상태 관리**: Flutter의 StatefulWidget을 활용한 상태 관리

### 🎯 기술 스택
- **Framework**: Flutter 3.32.6
- **언어**: Dart
- **상태 관리**: StatefulWidget + setState
- **로컬 저장소**: SharedPreferences
- **UI 라이브러리**: 
  - flutter_screenutil (반응형 디자인)
  - google_fonts (폰트)
  - lottie (애니메이션)
  - confetti (폭죽 효과)
  - fl_chart (차트)
  - table_calendar (캘린더)

## 앱 시연

### 📱 주요 화면 구성

#### 1. 홈 탭
- 연속 달성 일수 (스트릭) 표시
- 오늘의 일간 목표 목록
- 목표 완료 시 폭죽 애니메이션

#### 2. 목표 탭
- 월간/주간/일간 목표를 카테고리별로 표시
- 목표별 완료 상태 및 진행률 시각화
- 월간 목표 클릭 시 상세 페이지 이동

#### 3. 피드 탭
- 친구들의 공개 목표 및 회고 확인
- 좋아요, 댓글, 공유 기능
- 사용자 아바타 및 목표 타입별 색상 구분

#### 4. 회고 탭
- 날짜별 회고 그룹화
- 회고 타입별 아이콘 및 색상 구분
- 최신 회고 우선 정렬

#### 5. 프로필 탭
- 목표 완료율 통계
- 연속 달성 일수
- 최근 회고 목록

### 🎬 주요 기능 시연

#### 목표 생성 마법사
1. 목표 제목 및 설명 입력
2. 목표 시간 설정
3. 월간 목표 생성
4. 주차별 주간 목표 추가
5. 일간 목표 추가

#### 회고 작성
1. 목표 완료 시 회고 다이얼로그 표시
2. 회고 타입 선택 (한 줄/KPT/이모지)
3. 회고 내용 작성
4. 저장 시 폭죽 애니메이션 및 페이드아웃 효과

#### 소셜 상호작용
1. 친구 추가/삭제
2. 목표 공개 설정 변경
3. 피드에서 좋아요/댓글/공유
4. 댓글 페이지에서 실시간 댓글 작성

## WEGROW 특징과 이후의 개발 방향

### 🌟 WEGROW 특징

#### 1. 사용자 중심 설계
- **직관적 UX**: 복잡한 기능을 단순하고 직관적으로 구현
- **감정적 연결**: 폭죽 애니메이션 등으로 목표 달성의 기쁨을 전달
- **개인화**: 사용자별 맞춤형 목표 관리 시스템

#### 2. 체계적 목표 관리
- **계층적 구조**: 목표의 우선순위와 연관성을 명확히 제시
- **진행률 추적**: 시각적 피드백으로 동기부여 제공
- **현실적 목표 설정**: 시간 추적을 통한 데이터 기반 목표 설정

#### 3. 성장 중심 접근
- **회고 시스템**: 단순한 완료가 아닌 성찰을 통한 진정한 성장
- **다양한 회고 방식**: 사용자 성향에 맞는 회고 방법 제공
- **지속적 개선**: 과거 회고를 통한 목표 설정 개선

#### 4. 소셜 동기부여
- **적절한 경쟁**: 친구들과의 건강한 경쟁을 통한 동기부여
- **공개 설정**: 개인정보 보호와 소셜 기능의 균형
- **상호작용**: 단순한 관찰이 아닌 적극적 참여 유도

### 🚀 향후 개발 방향

#### 1. 고도화된 분석 기능
- **목표 달성 패턴 분석**: 사용자의 목표 달성 패턴 분석
- **시간대별 생산성 분석**: 언제 가장 효율적인지 파악
- **목표 난이도 조정**: 과거 데이터 기반 목표 난이도 자동 조정

#### 2. AI 기반 개인화
- **스마트 목표 추천**: 사용자 패턴 기반 목표 추천
- **회고 분석**: AI를 통한 회고 내용 분석 및 피드백
- **동기부여 메시지**: 상황별 맞춤형 격려 메시지

#### 3. 확장된 소셜 기능
- **목표 그룹**: 공통 목표를 가진 사용자들의 그룹 형성
- **멘토링 시스템**: 경험자와 초보자 간의 연결
- **챌린지**: 기간별 목표 달성 챌린지

#### 4. 통합 플랫폼
- **캘린더 연동**: Google Calendar, Apple Calendar 연동
- **할 일 관리**: 기존 할 일 관리 앱과의 연동
- **건강 데이터**: Apple Health, Google Fit 연동

  
### 💡 기술적 개선 방향

#### 1. 성능 최적화
- **상태 관리**: Provider/Riverpod 도입으로 상태 관리 개선
- **데이터베이스**: SQLite 도입으로 복잡한 쿼리 지원
- **캐싱**: 효율적인 데이터 캐싱 전략

#### 2. 사용자 경험 개선
- **오프라인 지원**: 네트워크 없이도 기본 기능 사용 가능
- **다크/라이트 모드**: 사용자 선호도에 따른 테마 변경
- **접근성**: 시각/청각 장애인을 위한 접근성 개선

#### 3. 보안 강화
- **데이터 암호화**: 민감한 데이터 암호화 저장
- **인증 시스템**: Firebase Auth 도입
- **개인정보 보호**: GDPR 준수

## 결론

WeGrow는 단순한 할 일 관리 앱을 넘어서, 사용자의 진정한 성장을 돕는 종합적인 목표 관리 플랫폼입니다. 계층적 목표 관리, 체계적인 회고 시스템, 그리고 소셜 기능을 통해 사용자가 지속 가능한 성장을 이룰 수 있도록 돕고자 합니다.

앞으로도 사용자 피드백을 바탕으로 지속적으로 개선해나가며, 더 많은 사람들이 의미 있는 목표를 세우고 달성할 수 있도록 지원하겠습니다.
